## 1. Dashboard Tanıtımı

```
UceAsistan Dashboard'a hoş geldiniz. 

Bu ekranda portföy performansınızı anlık olarak izleyebilirsiniz. Üst panelde toplam bakiye, varlık dağılımı ve günlük performans özeti yer almaktadır. 

Risk yönetimi göstergemiz, mevcut risk durumunuzu ve günlük limitinizi net bir şekilde sunar. Yeşil gösterge, güvenli bölgede olduğunuz anlamına gelmektedir. 

Alt bölümde işlem geçmişiniz listelenirken, sağ paneldeki yapay zeka asistanımız sorularınıza yanıt vermeye hazır beklemektedir.

Size en iyi deneyimi sunmak için buradayız.
```

---

## 2. Backtest Modülü

```
Strateji test modülümüzle tanışın.

Bu bölümde, trading stratejilerinizi geçmiş veriler üzerinde güvenle test edebilirsiniz. Sol menüden işlem yapacağınız sembolü ve zaman dilimini seçerek başlayın. 

Tarih aralığını belirledikten sonra, yapay zeka butonu ile otomatik strateji oluşturabilir veya kendi stratejinizi manuel olarak girebilirsiniz.

Test sonuçları grafik üzerinde görselleştirilir. Sonuç panelinde toplam getiri, başarı oranı ve maksimum düşüş gibi kritik performans metrikleri detaylı şekilde sunulmaktadır.

Bilinçli kararlar için veriye dayalı analiz yapmanızı öneriyoruz.
```

---

## 3. API Kurulumu

```
API kurulumunu birlikte tamamlayalım.

İlk olarak console.groq.com adresini ziyaret ederek ücretsiz hesabınızı oluşturun. Giriş yaptıktan sonra sol menüden API Keys bölümüne geçin.

Create API Key butonuna tıklayarak yeni anahtarınızı oluşturun ve güvenli bir şekilde kopyalayın. Bu anahtarı yalnızca bir kez görüntüleyebileceğinizi hatırlatmak isteriz.

UceAsistan'a dönerek Ayarlar bölümünden API anahtarınızı ilgili alana yapıştırın ve kaydedin.

Kurulum tamamlandı. Artık yapay zeka asistanınız tüm özellikleriyle hizmetinizde.
```

---

## 4. AI Asistan Kullanımı

```
Yapay zeka asistanınız her an yanınızda.

Sohbet paneline sorularınızı yazarak piyasa analizi, strateji önerileri veya risk yönetimi konularında destek alabilirsiniz. Asistanımız, sorularınızı değerlendirerek size özel ve detaylı yanıtlar sunmaktadır.

Geçmiş konuşmalarınız sol panelde arşivlenmektedir. Dilediğiniz zaman önceki analizlerinize geri dönebilirsiniz.

Profesyonel tavsiyemiz: Sorularınızı ne kadar net ve spesifik ifade ederseniz, o kadar kapsamlı yanıtlar alırsınız.

Başarılı işlemler dileriz.
```
